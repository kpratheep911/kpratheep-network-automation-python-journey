# Project 001 – Device Inventory Reader

## Overview

The **Device Inventory Reader** is a Python application that reads a network device inventory from a CSV file and displays device information in a clear, structured format.

This project is the first step in my **Network Automation Python Journey**, where I build practical automation tools for network engineers.

## Objectives

* Read a CSV inventory file using Python
* Display network device information
* Practice working with tabular data using Pandas
* Build a reusable foundation for future automation projects

## Technologies Used

* Python 3
* Pandas

## Project Structure


## Features

* Load a CSV inventory file
* Display the total number of devices
* Display device details in a readable format
* Handle large inventory files efficiently
* Provide a reusable code base for future projects
